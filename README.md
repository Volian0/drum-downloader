# drum-downloader
Drum Downloader for Piano Tiles 2
# How to install
`git clone https://github.com/Volian0/drum-downloader`  
`cd drum-downloader`  
`g++ src/*.cpp -std=c++11 -lurlmon -o drum_downloader`  
# How to update
`cd drum-downloader`  
`git pull`  
`g++ src/*.cpp -std=c++11 -lurlmon -o drum_downloader`
